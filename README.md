# HTML Builder
PHPでHTMLの組み立てができるライブラリ

#### autoloadの作成

```
$ make setup
```

#### ローカルサーバの起動

```
$ make up
```

#### 各ファイルの説明

|ファイル名|説明|
|:--|:--|
|HtmlBuilder|要素とポインタの管理, 各トレイトを呼び出すクラス|
|Root|根本となる要素のトレイト|
|Meta|文書のメタデータ関連のトレイト|
|Section|セクション関連のトレイト|
|Content|コンテンツの分類関連のトレイト|
|Text|テキスト関連のトレイト|
|Embedded|コンテンツの埋め込み関連のトレイト|
|Table|テーブル関連のトレイト|
|Form|フォーム関連のトレイト|
